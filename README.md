# Highrise Bot Node.js

Бот для игры Highrise, написанный на Node.js с использованием `highrise-js-sdk`.

## Функционал
- Приветствие игроков при входе.
- Команды: `!ping`, `!help`, `!about`.
- Логирование чата в консоль.

## Установка и запуск

1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` в корневой папке и добавьте ваши данные:
   ```env
   HIGHRISE_TOKEN=ваш_токен
   HIGHRISE_ROOM_ID=id_комнаты
   ```
4. Запустите бота:
   ```bash
   npm start
   ```

## Деплой
Для деплоя рекомендуется использовать платформы, поддерживающие долгоживущие процессы (Render, Railway, VPS), так как бот использует WebSocket соединение.
