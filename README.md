# Highrise Bot Node.js

Бот для игры Highrise, написанный на Node.js с использованием `highrise.sdk.dev`.

## Функционал

- **Авто-приветствие** игроков при входе в комнату
- **Авто-повтор эмоций** — бот повторяет эмоции других игроков
- **Реакция на обращения** — отвечает когда кто-то пишет "бот" или "bot"
- **Hot Reload** — автоматическая перезагрузка команд при изменении файлов
- **Логирование** — весь чат и события выводятся в консоль

## Команды

| Команда   | Описание                          |
| --------- | --------------------------------- |
| `!ping`   | Проверка работоспособности бота   |
| `!users`  | Количество игроков в комнате      |
| `!about`  | Информация о боте                |
| `!help`   | Список доступных команд          |

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Создайте файл `.env` в корневой папке:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ROOM_ID=id_комнаты
   ```

3. Запустите бота:
   ```bash
   npm start
   ```

## Деплой

Для деплоя рекомендуется использовать платформы с поддержкой долгоживущих процессов (Render, Railway, VPS), так как бот использует WebSocket соединение и автоматически переподключается при разрыве.

## Структура проекта

```
src/
├── CommandHandler.js   # Обработчик команд с Hot Reload
└── commands/
    ├── about.js       # Команда !about
    ├── help.js        # Команда !help
    ├── ping.js        # Команда !ping
    └── users.js       # Команда !users
```

## Добавление новых команд

Создайте файл в `src/commands/` по образцу:

```javascript
module.exports = {
  name: "команда",
  description: "Описание команды",
  async execute(bot, user, message, args) {
    bot.chat.send("Результат");
  }
};
```

Команда автоматически загрузится и будет доступна после Hot Reload.
